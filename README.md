# IP Info Auto Fill

基于IP地理位置自动生成并填写网页注册表单的浏览器插件。

## ✨ 功能特性

- 🌍 **智能获取IP地理位置** - 使用 ip-api.com 获取当前IP的国家、城市信息
- 🎲 **随机信息生成** - 根据地理位置生成匹配的姓名、用户名、邮箱等
- ✏️ **可编辑修改** - 生成的信息可以手动修改
- 🔄 **单独刷新** - 每个字段都可以单独重新生成
- ✍️ **一键填写** - 自动识别并填写常见表单字段

## 📦 支持的信息类型

| 字段 | 说明 |
|------|------|
| 名/姓 | 根据国家生成对应语言的姓名 |
| 用户名 | 基于姓名生成多种风格的用户名 |
| 邮箱 | 用户名 + 常见邮箱域名 |
| 密码 | 12位强密码（包含大小写、数字、符号）|
| 电话 | 带国家区号的电话号码 |
| 地址 | 随机街道地址 |
| 城市 | 来自IP定位的城市 |
| 邮编 | 根据国家格式生成 |
| 国家 | 来自IP定位 |

## 🚀 安装方法

### Chrome / Edge

1. 打开扩展程序页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹

### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击「临时加载附加组件」
3. 选择项目中的 `manifest.json` 文件

## 📖 使用方法

1. 打开任意注册页面
2. 点击浏览器工具栏中的插件图标
3. 查看自动生成的信息（可手动修改）
4. 点击「填写表单」按钮

## 🌐 支持的语言/地区

插件会根据IP所在国家自动选择对应的名字库：

- 🇺🇸 英语国家（美国、英国、加拿大、澳大利亚）
- 🇨🇳 中文地区（中国、台湾、香港、新加坡）
- 🇯🇵 日本
- 🇰🇷 韩国
- 🇩🇪 德语国家（德国、奥地利、瑞士）
- 🇫🇷 法语国家（法国、比利时）
- 🇷🇺 俄罗斯
- 🇪🇸 西班牙语国家
